= pinus-flexilis

CLI simulator for Agathis Management Controller.

== Build

[source,bash]
----
mkdir build
cd build
cmake ..
make
----

== Usage

[source,bash]
----
./pinus-flexilis
----

To exit CTRL+C

You can open multiple instances of the program and they will start communicating between them to simulate wireless connection.
